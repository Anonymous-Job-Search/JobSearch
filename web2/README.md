# Anonymous Job Search - WEB

## Development

```shell
# Run/Stop project
make up
make down

# Show logs
make logs

# Restart web container
make restart

# Open docker container
make sh

# Production build
make build

# Run linters
make lint
make typecheck
make eslint
make prettier
make stylelint

# Run tests
make test
make test-watch
make test-coverage

# Execute pnpm binary inside a container
./pnpm [command]
```
