# Anonymous Job Search - WEB

## Development

```shell
# Run/Stop project
make up
make down

# Show logs
make logs

# Open docker container
make sh
# Production build
make build
# Run linters
make lint
make typescript
make eslint
make prettier
make stylelint

# Run tests
make test
make test-coverage
make test-unit
make test-unit-coverage
make test-unit-watch

# Execute pnpm binary inside a container
./pnpm [command]
```
